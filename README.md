# pnpm-ts-react-example

## Usage

```sh
npm install pnpm-ts-react-example
```

## Development

Install pnpm v7

### Setup

```sh
git clone 
pnpm install
```

### Run demo

```sh
pnpm start
```

### Lint and format

```sh
pnpm lint
pnpm format
```

### Build library

```sh
pnpm build
```

### Build demo

```sh
pnpm build-demo
```

### Monorepo tasks

```sh
pnpm meta-updater
```
